# 下载与安装

## 安装前说明

要想下载LavaArcade，你的设备需要满足如下配置项：

|类别|最低配置|推荐配置|
| -------------------------| ----------------------------| ------------------------------------|
|预留的存储空间|1GB|2GB|
|CPU|2 核心 4 线程，主频 2.0GHz|4 核心 8 线程，主频 2.5GHz|
|设备内存|6GB|8GB~12GB|
|显卡（若需加速，可选）|-|NVIDIA GTX 1060、AMD RX 580 或更高|
|设备架构|x64（amd64）或Arm64|x64（amd64）或Arm64|

## 下载

目前支持以下渠道下载正式版或预览版：

Github Releases：[https://github.com/qwq672/LavaArcade/releases](https://github.com/qwq672/LavaArcade/releases)

蓝奏云（可能有缺失，密码0000）：[https://xiaochat.lanzouu.com/b0xxd3t6d](https://xiaochat.lanzouu.com/b0xxd3t6d)

目前仍在开发LavaArcade，对于完整功能请敬请期待。你可以下载预览版来体验版本中增加或修复的功能。

## 安装

请确保需要安装的版本满足以下要求：

|类别|要求|
| -----------------| -----------------------------|
|Java版本|Java 17（64位）|
|Minecraft 版本|Java 版 1.20.1|
|模组加载器|Fabric|
|Fabric 版本|0.15.0 或更高|
|Fabric API 版本|0.92.0 或更高|
|前置模组|Carpet|
|前置模组版本|1.20-1.4.112+v230608 或更高|

检查完并下载完模组jar文件后，

将jar文件（LavaArcade-vXXXXXX.jar）放入./minecraft/mods里（如果未开启版本隔离）

或放入./minecraft/[版本号]/mods里（如果开启版本隔离）

放入后，即可完成安装。随后，照常使用启动器启动即可。

## 有关优化模组

强烈建议安装优化模组。对于低配置的设备，此可选选项是最佳选择。安装适量的优化模组可以提升你的游玩体验并降低资源占用。
